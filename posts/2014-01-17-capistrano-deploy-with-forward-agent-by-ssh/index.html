Capistrano Deploy With Forward Agent By SSH

使用Capistrano部署应用的时候，服务器往往会到一个具有访问权限的git仓库拉取代码; 如果你不想把 server 的 pub key，copy 到仓库服务器上的话，你可以尝试使用本地代理的方式。

具体操作如下：

####1. 配置deploy文件,手动开启代理服务.

set :ssh_options, {:forward_agent => true}

####2. 告诉SSH Agent你的key

$ ssh-add ~/.ssh/id_rsa
